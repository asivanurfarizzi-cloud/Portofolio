## ✨ Features
* **Modern UI/UX:** Clean design with glassmorphism elements and smooth scrolling.
* **Dark/Light Mode:** Integrated theme toggle saving user preference via `localStorage`.
* **Interactive Animations:** Features typing effects, 3D tilt effects on project cards, and scroll reveal animations.
* **Fully Responsive:** Optimized for both desktop and mobile viewing.
* **Direct Contact:** Integrated direct WhatsApp and email links for easy communication.

## 🛠️ Tech Stack
* HTML5
* CSS3 (Custom Variables, Flexbox, CSS Grid)
* Vanilla JavaScript (DOM Manipulation, IntersectionObserver)
